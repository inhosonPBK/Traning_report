서비스 소개

Weekly Training Report란 무엇인가

📌 서비스 목적

Weekly Training Report는 Promega Korea 인턴의 주간 교육 훈련 내용을 체계적으로 기록하고 관리하기 위한 내부 웹 서비스입니다. 인턴은 매주 학습 내용을 작성하고 제출하며, 멘토는 피드백을 남기고, 매니저는 전체 현황을 한눈에 파악할 수 있습니다.

👥 역할(Role) 구조
역할 설명 주요 기능
Intern 교육 훈련 중인 인턴 주간 레포트 작성 · 제출 · 회수
Mentor 인턴을 담당하는 멘토 제출된 레포트 검토 · 피드백 작성 · 완료 처리
Manager 팀 전체 레포트를 열람하는 매니저 모든 인턴의 레포트 현황 조회
Admin 시스템 관리자 (역할과 별개로 부여) 사용자 승인 · 역할 부여 · 비밀번호 초기화
💡 Admin은 역할이 아닌 권한입니다. 예를 들어 Mentor이면서 동시에 Admin 권한을 가질 수 있습니다. Admin 권한이 있는 사용자는 상단 내비게이션에 Admin 버튼이 추가로 표시됩니다.
🔄 전체 워크플로우
1
회원 가입 요청

신규 사용자가 /signup 페이지에서 이름·이메일·비밀번호·부서를 입력하고 가입 신청합니다.

2
어드민 승인

Admin 권한 보유자가 Admin Panel에서 신청 내역을 확인하고 역할을 지정한 뒤 승인합니다.

3
인턴 레포트 작성 & 제출

승인된 인턴이 로그인 후 해당 주차 레포트를 작성하고 제출합니다.

4
멘토 리뷰 & 완료

담당 멘토가 제출된 레포트를 확인하고 피드백을 작성한 뒤 완료(Complete Review) 처리합니다.

5
매니저 현황 조회

매니저가 All Reports 화면에서 전체 인턴의 레포트 제출 현황과 완료된 레포트를 열람합니다.
